# Online Food Ordering System

## Project Overview

The Online Food Ordering System is a Java-based application developed to simplify the process of ordering food online. This system allows customers to browse available food items, add products to a cart, calculate the total bill, and place orders efficiently. The project demonstrates the use of Java programming concepts such as classes, objects, arrays, collections, methods, conditional statements, loops, and user input handling.

The primary objective of this project is to provide a simple and user-friendly platform where customers can order food digitally instead of using traditional ordering methods. The system improves convenience, reduces manual effort, and helps users manage their food orders effectively.

---

## Features

### User Login

The system includes a secure login module where users must enter a valid username and password to access the application. This ensures that only authorized users can place orders.

### Food Menu

Customers can view a list of available food items along with their prices. The menu includes popular food items such as:

* Chicken Biryani
* Pizza
* Burger
* Shawarma
* Fried Rice
* Soft Drinks

### Add to Cart

Users can select food items from the menu and add them to their shopping cart. Multiple items can be added during a single session.

### Cart Management

The cart stores all selected food items and displays the current order details. Users can review their selected items before confirming the order.

### Bill Calculation

The application automatically calculates the total bill based on the items selected by the customer.

### Customer Information

The system collects customer details including:

* Customer Name
* Phone Number
* Delivery Address

These details are used for order confirmation and delivery purposes.

### Order Placement

Once the customer confirms the order, the system generates an order summary displaying:

* Customer Information
* Ordered Items
* Total Amount
* Delivery Information

### Order Confirmation

After successful order placement, the system displays a confirmation message along with the estimated delivery time.

## System Workflow

1. User launches the application.
2. User enters login credentials.
3. Customer details are collected.
4. Food menu is displayed.
5. User selects food items.
6. Selected items are added to the cart.
7. Cart details are displayed.
8. Total bill is calculated.
9. Customer places the order.
10. Order summary and confirmation are generated.
